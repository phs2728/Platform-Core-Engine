# Creative Knowledge Engine

> **AI Research Platform** — The brain of the creative pipeline. Research First, Evidence-Based.

## Purpose

This engine performs what world-class UX agencies do before starting any project:

```
Client Interview → Business Research → Market Research → Competitor Research →
Existing Website Audit → UX/SEO/A11y/Performance/Content Audits →
Pattern Extraction → Creative Brief → Creative Intelligence Engine
```

**AI never starts with design. It starts with Research.**

## Acceptance

> *"Creative Knowledge Engine을 삭제하면 플랫폼의 Research, Competitor Analysis, Website Audit, Knowledge Base, Creative Brief 생성 기능이 모두 사라지는가?"*
>
> **YES**

## Evidence Engine

Every recommendation MUST include:

```
Recommendation → Evidence → Confidence → Sources → Reason → Expected Impact
```

AI cannot make claims without evidence.

## Provider Plugin Architecture

No browser/crawler/scraper logic. All data gathering through Host Interfaces:

| Provider | Purpose |
|---|---|
| `IWebsiteAuditProvider` | Provides website audit data |
| `ICompetitorProvider` | Provides competitor analysis data |
| `IScreenshotProvider` | Provides visual snapshots |
| `IHTMLProvider` | Provides HTML structure |
| `ISEOProvider` | Provides SEO metrics |
| `IPerformanceProvider` | Provides performance metrics |
| `IAccessibilityProvider` | Provides accessibility metrics |
| `IResearchProvider` | Provides market/industry research data |
| `ILLMProvider` | Provides LLM analysis capabilities |

## Absolute Principles

- **Research First** — no design without research
- **Evidence Required** — no recommendation without evidence
- **No browser/crawler/scraper** — Provider Plugin only
- **No Business Logic** — research & knowledge only
- **Industry Agnostic**
- **Constitution C-1 ~ C-23** compliance

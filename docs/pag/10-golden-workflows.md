# PAG Part 10 — Golden Workflows

> Complete workflows for 11 industries. Deterministic. Reproducible.

## Universal Workflow Template

Every industry follows this structure:

```
1. Client Interview → Industry + Sub-type classification
2. Playbook Selection (deterministic)
3. Skill Pack Selection (deterministic)
4. Agency OS → Swarm Creation
5. Research Swarm → Customer Psychology + Competitor Analysis
6. Creative Swarm → Art Direction + Brand Voice + Copy Framework
7. UX Swarm → CDA + Trust Architecture + Detail Strategy
8. Engineering Swarm → Theme → Component → CMS → Studio
9. Frontend Generation (React/Next.js)
10. QES Assessment → PASS/WARNING/FAIL
11. Fix → Re-assess → Release
12. Learning Feedback
```

## Industry-Specific Variations

### Restaurant
- **Playbook**: `playbook-restaurant`
- **Key Trust Evidence**: Real food photos, chef face, origin, live reservation
- **Detail Page**: Menu Detail (8 sections: Hero Photo → Description → Ingredients → Nutrition → Chef Story → Reviews → Price → Related)
- **Objections**: Price, vegetarian, allergy, parking, kids
- **Primary CTA**: "Reserve a Table" / "Order Now"
- **Photo Direction**: High-res food photography, multiple angles, no stock
- **QES Minimum**: Premium

### Hotel (Luxury)
- **Playbook**: `playbook-luxury-hotel`
- **Key Trust Evidence**: Real rooms, verified reviews, best price guarantee, awards, since year
- **Detail Page**: Room Detail (7 sections: Gallery → Details → Amenities → Policies → Availability → Reviews → Book)
- **Objections**: Safety, late check-in, WiFi, breakfast, airport pickup
- **Primary CTA**: "Book Now"
- **Photo Direction**: Room multi-angle + bathroom + view, full-bleed hero
- **Art Direction**: Luxury (extreme whitespace, restrained motion, editorial typography)
- **QES Minimum**: Agency Grade

### Hotel (Boutique Hostel)
- **Playbook**: `playbook-boutique-hostel`
- **Key Trust Evidence**: Community photos, guest reviews, social atmosphere, safety
- **Objections**: Safety, women solo, late check-in, WiFi, towels, airport pickup
- **Primary CTA**: "Check Availability"
- **Photo Direction**: Community/common areas, guest photos, social vibe
- **QES Minimum**: Premium

### Travel
- **Playbook**: `playbook-travel-agency`
- **Key Trust Evidence**: Local operation, real guide, tour photos, emergency contact, partners
- **Detail Page**: Tour Detail (8 sections: Gallery → Timeline → Map → Guide → Included → Reviews → FAQ → Book)
- **Objections**: Refund, language, kids, difficulty
- **Primary CTA**: "Book Tour"
- **QES Minimum**: Premium

### Marketplace
